# The Evolution of Todo - Project Overview

## Project Theme
From CLI to Distributed Cloud-Native AI Systems

## Project Goal
Students act as Product Architects using AI + Spec-Driven Development (no manual boilerplate coding).

## Phase Evolution
- **Phase I**: In-memory Python console application
- **Phase II**: Full-stack web application with Next.js + FastAPI
- **Phase III**: Chatbot integration (planned)

## Current Status
Both Phase I and Phase II are completed and functional.
